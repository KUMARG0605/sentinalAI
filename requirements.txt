#  POST-INSTALL STEPS
#
#  1. Install Playwright browsers:
#       python -m playwright install chromium
#
#  2. Download Vosk model for wake word (offline):
#       Place in: resources/vosk_models/vosk-model-en-in-0.5/
#       Download: https://alphacephei.com/vosk/models
#       Recommended: vosk-model-en-in-0.5 (~40MB, Indian English)
#
#  3. Copy resources/ folder to project root:
#       resources/
#         piper_models/piper.exe     ← TTS engine
#         piper_models/models/...    ← voice model files
#         vosk_models/...            ← offline wake word model
#
#  4. Set API keys in .env:
#       GROQ_API_KEY=gsk_...
#       ASSEMBLYAI_API_KEY=...
#       TAVILY_API_KEY=...           (optional)
#       SAMBANOVA_API_KEY=...        (optional fallback)
